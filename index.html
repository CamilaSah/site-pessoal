Sayuri Tokubo

Desenvolvedora Front-end júnior

Em 2023, decidi migrar de carreira da engenharia de produção para o desenvolvimento front-end.

Essa decisão veio a partir das experiências anteriores que tive, que me mostraram o meu interesse pela melhoria de processos focados no usuário. A princípio, acreditava que a área de UX/UI design seria a mais adequada para mim, até entender que o meu perfil mais analítico, lógico, detalhista se encaixaria melhor no desenvolvimento front-end, onde eu poderia desenvolver soluções adequadas para o usuário final.

Meu foco de estudos atualmente é em React. Fiz duas formações na Alura para aprender mais profundamente a lógica de programação, HTML, CSS e Javascript e o meu primeiro resultado é esse site pessoal.
